NightCity

NightCity is a single player 3-D browser based game that provides an immersive experience by allowing the user to interact with the game using their hands as if they were using an actualy steering wheel to drive around NightCity. Inspired by the following music video, the goal of the game is to get up to 20 points by dodging the demon to win. The player loses if they lose 3 consecutive points in a row.

Instructions

  • Position the camera so that the user is facing the road.
  • The user must use their hands to steer the car and avoid the demon.
  • The user must avoid the demon at all costs or else they will lose a point.
  • The game ends when three consecutive points are lost (loss) or if the score is greater than 20 (win).
